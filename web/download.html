bsteeleMusicApp Download Page

Web Version

The bsteeleMusicApp is usually found on the web at bsteeleMusicApp. This web version will be the most universal and easiest to use of the application formats.

Native Versions

Native applications are also available. They are more difficult to install but will typically provide slightly better performance. Song list and song history updates are provided in the cloud so native applications should have access to the web for the best user experience.

The following can be downloaded using the these links:

    The Macos will not be happy about running an app downloaded from the web, but you can force it with: Apple icon, System Preferences, Security & Privacy, "Allow apps downloaded from:", select the app to enable it. Source Code

The application is written in Flutter and Dart. See Flutter. The source code is available on github.com in three separate projects at:

  • bsteele_music_flutter: This is the GUI section of the application written using the data model in the project bsteeleMusicLib. This split facilitates the logical division of functionality, provides a command line interface independent of flutter and increases the speed of testing.
  • bsteeleMusicLib: This is the data model support for the application. This project is intentionally strictly dart.
  • bsteeleMusicServer: This is a small Tomcat9 server for the leader/follower feature. Note that it's not available for Tomcat10 and that it has been written in Java. The dart server interfaces do not easily allow cross websocket connection communication like the java one does.

Technical Notes:

  • The build scripts in the bsteele_music_flutter/bin directory are tailored to my development environment and are not likely but be of much use otherwise. The projects should build and test properly from the usual flutter and dart command line interface.
  • The bsteele_music_flutter/pubspec.yaml assumes that the bsteeleMusicLib project is in an adjacent directory.
