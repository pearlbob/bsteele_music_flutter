bsteeleMusicApp Download Page

The bsteeleMusicApp is usually found on the web at bsteeleMusicApp.

The app can also be downloaded at the following links:

  • Macos x86 dmg: bsteeleMusicApp.dmg. After downloading, verify the sha256sum as:

    The Macos will not be happy about running an app downloaded from the web but you can force it with: Apple icon, System Preferences, Security & Privacy, "Allow apps downloaded from:", select the app to enable it. When re-installing the app, you will again have to force acceptance.

    Once installed, the app can be found in the system Applications folder labeled "bsteele_music_flutter.app".

  • Android apk: bsteeleMusicApp.apk. The sha256sum as:

    Android will not willingly download and install apps from the web. Please research this and understand your security exposure. I suggest you start with androidauthority.com.

  • Raspberry Pi aarch64: bsteeleMusicAppPi.tgz. The sha256sum as:

    This tar file needs to be extracted to a convenient directory in a Raspberry Pi with a 64 bit OS. Run the binary bsteele_music_flutter can be run there. Note that it requires the lib and data directories be in their relative positions. Performance on a Pi 3 is marginal. Tested on the Debian 11, bullseye OS. Song and singer updates will come from the web so assess is encouraged.

  • Apache Tomcat war file (a Java web archive file): bsteeleMusicApp.war. After downloading, verify the sha256sum as:

    The war file contents can also be extracted to a webserver (typically Apache) and served as static webpages without the leader/follower function.
